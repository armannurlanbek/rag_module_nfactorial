### Студент 2: Chunking Strategy 1 — Semantic Coherence

#### Задача

Оценить семантическую связность чанков используя векторные представления.

#### Методология

**Метрика: Semantic Coherence Score**

Идея: если предложения в чанке семантически связаны, их векторы должны быть близки друг к другу.

```python
import numpy as np
from sentence_transformers import SentenceTransformer
from sklearn.metrics.pairwise import cosine_similarity

model = SentenceTransformer("all-MiniLM-L6-v2")

def semantic_coherence(sentences):
    # Получаем эмбеддинги
    embeddings = model.encode(sentences)

    # Считаем попарные cosine similarities
    similarities = cosine_similarity(embeddings)

    # Берём только верхний треугольник матрицы (без диагонали)
    upper_triangle = similarities[np.triu_indices_from(similarities, k=1)]

    # Среднее значение схожести = наш score
    coherence_score = upper_triangle.mean()

    # Дополнительно: смотрим на дисперсию
    coherence_variance = upper_triangle.var()

    return {
        "coherence_score": float(coherence_score),
        "coherence_variance": float(coherence_variance),
        "num_sentences": len(sentences)
    }


# Пример использования
chunk = """
Машинное обучение - это раздел искусственного интеллекта.
Оно позволяет компьютерам обучаться на данных.
Существует множество алгоритмов машинного обучения.
"""
sentences = [s.strip() for s in chunk.split(".") if s.strip()]
result = semantic_coherence(sentences)
```

1. **Реализацию** метрики Semantic Coherence Score
2. **Таблицу** сравнения стратегий чанкирования

   ```
   | Стратегия       | coherence_score | variance | num_sentences |
   | --------------- | --------------- | -------- | ------------- |
   | fixed_size      | ...             | ...      | 120           |
   | ...             | ...             | ...      | ...           |
   ```

3. **Визуализации** (графики распределения coherence scores)
4. **Рекомендация:** какая стратегия даёт наиболее связные чанки

### Студент 3: Chunking Strategy 2 — Context Independence (LLM as a Judge)

#### Задача

Оценить самодостаточность чанков используя LLM как судью.

#### Методология

**Метрика: Context Independence Score**

Идея: хороший чанк должен быть понятен без дополнительного контекста.

```python
import openai
# или использовать любую другую LLM через API

def evaluate_context_independence(chunk_text, llm_model="gpt-4o-mini"):
    """
    Оценивает, насколько чанк самодостаточен и понятен без контекста.

    Returns:
        score: int (1-5)
        reasoning: str
    """
    prompt = f"""
Оцени следующий текстовый отрывок по шкале от 1 до 5, где:

**5** - Отрывок полностью самодостаточен. Можно понять суть без дополнительного контекста.
**4** - В основном понятен, но есть 1-2 неясных референса.
**3** - Частично понятен, но требуется контекст для полного понимания.
**2** - Смысл неясен без предыдущего/следующего текста.
**1** - Отрывок бессмысленен без контекста (обрывки предложений, референсы на "это", "такой" без пояснения).

**Критерии оценки:**
- Есть ли полные предложения или обрывки?
- Используются ли местоимения без антецедентов ("это", "он", "такой")?
- Можно ли понять основную идею отрывка?
- Есть ли введение/контекст для терминов?

**Отрывок:**
{chunk_text}

**Ответ дай в формате JSON:**
{{
  "score": X,
  "reasoning": "Краткое объяснение оценки",
  "issues": ["список проблем с контекстом, если есть"]
}}
"""
    response = openai.ChatCompletion.create(
        model=llm_model,
        messages=[
            {"role": "system", "content": "Ты эксперт по оценке качества текста."},
            {"role": "user", "content": prompt}
        ],
        temperature=0.3
    )
    return response.choices[0].message.content


# Пример использования
chunk = """
Это позволяет значительно улучшить результаты.
Метод был предложен в предыдущей работе.
"""
result = evaluate_context_independence(chunk)
```

#### Что предоставить

1. **Реализацию** метрики Context Independence Score
2. **Таблицу** сравнения стратегий чанкирования

   ```
   | Стратегия       | avg_score | min_score | max_score |
   | --------------- | --------- | --------- | --------- |
   | fixed_size      | ...       | ...       | ...       |
   | ...             | ...       | ...       | ...       |
   ```

3. **Примеры** хороших и плохих чанков с объяснениями от LLM
4. **Рекомендация:** какая стратегия даёт наиболее самодостаточные чанки
